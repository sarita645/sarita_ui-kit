複製 css 色票 (使用前，記得更新)

使用說明：較易變化(換色) 。貼至 .css 檔案內，放置順序為最下面。

也可直接參考色碼使用。

文字已成功複製！

方法1. 在CSS內使用(可利用定義)

在 CSS 中，定義於 :root 中的變數可以用於任何 CSS 屬性中，並且可以在 HTML 的 class 中間接引用。

1.定義變數
使用 :root 定義全域 CSS 變數：

:root {
--red-50: #ffebee;
--red-100: #ffcdd2;
--red-200: #ef9a9a;
}

2.在 CSS 中使用變數

.example {
background-color: var(--red-100);
color: var(--red-900);
border: 1px solid var(--red-200);
}

3.在 HTML 中間接引用

填入class名稱，class="example" 即可引用。

方法2. 直接使用色碼

不確定是否會在更新，可複製的css色票內有對應的色票號碼。
