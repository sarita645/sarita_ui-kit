// 變數定義
$border-style: solid;
$border-width: 0px;
$padding: 0px;
$display: flex;
$align-items: center;
$justify-content: center;
$margin: 2px 0;
$line-height: 1.5;
$default-font-size: 16px;
$font-family: "NotoSansTc-Regular", sans-serif;
$color: #333333;

// 混合，用於生成文字樣式
@mixin text-style($font-size: $default-font-size, $font-weight: normal) {
border-style: $border-style;
border-width: $border-width;
font-size: $font-size;
font-weight: $font-weight;
padding: $padding;
// display: $display;
// align-items: $align-items;
// justify-content: $justify-content;
margin: $margin;
line-height: $line-height;
font-family: $font-family;
color: $color;
}

關於留白與間距
標題與內文的間距

標題若為H1 接續內文需距離24px

標題 標題與內文之間

內文的間距

使用8px作為間距的基礎。
大尺寸之間的間距還須依照倍率下去做調整。

spacing unit × ½ = 4px
spacing unit × 1 = 8px
spacing unit × 2 = 16px
spacing unit × 3 = 24px
spacing unit × 5 = 40px
spacing unit × 8 = 64px
spacing unit × 13 = 104px
